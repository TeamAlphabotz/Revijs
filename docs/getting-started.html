Getting Started

ReviJs converts your React/Vite SPA into static HTML using a headless browser. Set up in under 30 seconds — no cloud, no backend, no rewrites required.

Requirements

Before installing, make sure your environment meets these requirements:

  • ✅  Node.js v18.0.0 or higher
  • ✅  npm v9+ (or yarn / pnpm)
  • ✅  A built SPA — React, Vite, Vue, Svelte, etc.
  • ✅  OS — macOS, Linux, or Windows

Installation

Install ReviJs into your project. The browser engine (Playwright Chromium) installs automatically as part of the postinstall step — you don't need to do anything extra.

terminal
# npm
npm install @revijs/core

# yarn
yarn add @revijs/core

# pnpm
pnpm add @revijs/core
ℹ️ Auto Setup

Playwright Chromium and all required system dependencies install automatically. You never need to run playwright install manually.

Quick Start

Follow these steps to prerender your first SPA:

  1. Build your SPA

    Run your normal build command. ReviJs works on top of the dist/ output.

  2. Create a config file

    Run npx revijs init to auto-generate a starter config, or create it manually.

  3. Run the prerenderer

    Run npx revijs. It spins up a local server, opens each route in a headless browser, and saves the HTML.

  4. Deploy the output

    Upload dist-prerendered/ to your server. Use the optional middleware to serve it to bots automatically.

terminal
# Step 1: build your app
npm run build

# Step 2: generate config
npx revijs init

# Step 3: prerender all routes
npx revijs

⚠️ Build First

Always run npm run build before npx revijs. ReviJs renders your built output, not your source files.
