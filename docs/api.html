Programmatic API

Use ReviJs directly in your own Node.js scripts instead of the CLI. Useful for custom build pipelines, CI workflows, or integrating with other tools.

prerender()

prerender(config)
async

The main entry point. Renders all routes defined in config and writes static HTML to disk. This is the programmatic equivalent of running npx revijs.

import { prerender } from '@revijs/core';

await prerender({
  routes: string[],
  outputDir: string,
  distDir: string,
  engine: 'browser' | 'advanced' | 'ssr',
  waitFor: number,
  headless: boolean,
});
ParamTypeDescription
routesstring[]Routes to prerender. Required.
outputDirstringOutput directory for static HTML files.
distDirstringYour built SPA directory.
enginestringRendering engine to use.
waitFornumberms to wait after network idle.
headlessbooleanRun browser headless.
prerender-script.js
import { prerender } from '@revijs/core';

await prerender({
  routes: ['/', '/about', '/contact'],
  outputDir: 'dist-prerendered',
  distDir: 'dist',
  waitFor: 1500,
});

// run with: node prerender-script.js

renderAllPages()

renderAllPages(config)
async

Lower-level function that accepts a fully resolved config object. Starts the server, boots the engine, renders all routes, then tears everything down.

import { renderAllPages, loadConfig } from '@revijs/core';

const config = await loadConfig('revi.config.js');
await renderAllPages(config);

loadConfig()

loadConfig(path?, overrides?)
async

Loads and validates revi.config.js, merges with defaults and CLI overrides. Returns a fully resolved config object.

import { loadConfig } from '@revijs/core';

const config = await loadConfig('revi.config.js', {
  debug: true,
});

isBot() / detectBot()

isBot(userAgent)
sync

Returns true if the given user-agent string belongs to a known crawler. Used internally by the middleware.

import { isBot, detectBot } from '@revijs/core';

isBot('Googlebot/2.1'); // true
isBot('Mozilla/5.0 ...'); // false
detectBot('GPTBot/1.0'); // 'gptbot'
